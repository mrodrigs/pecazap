# PeçaZap - Teste

## Instruções para iniciar o projeto

**Requerimentos:**

* Versão atualizada do NodeJS instalada.

1.Clone o repositório do projeto:

`git clone https://github.com/mrodrigs/pecazap.git`

2.Instale as dependências do projeto usando o comando:

`yarn`

3.Inicie o projeto:

`yarn start`
